# Setu Groups Landing Website

Production-ready Next.js 14 App Router landing site for Setu Groups.

## Stack

- Next.js 14
- React 18
- TypeScript
- TailwindCSS
- Framer Motion

## Routes

- `/` Home
- `/setu-track` Setu Track product page
- `/pricing` Pricing page
- `/setu-crm` Setu CRM waitlist

## Local setup

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## Deploy to Vercel

1. Push the repository to GitHub.
2. Import the repo into Vercel.
3. Framework preset: Next.js.
4. Deploy.

## Notes

- Replace `hello@setugroups.com` with your real email.
- Replace placeholder client logos as needed.
- Add your live Setu Track URL if you want the CTA to point outside the site.
